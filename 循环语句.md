# for循环
1.  格式
	1. for(初始化;条件判断语句;条件控制语句){循环体语句}
# while循环语句
1. while(条件判断句){循环体语句；条件控制语句；}
2. 先执行语句再做判断
# do……while循环语句
1. do{循环体控制语句;条件控制语句}while（判断语句）;
2. 无论是否满足都会先执行一次，再进行判断
# 跳转循环语句
1. break：用于结束循环或switch语句
2. continue：用于跳过当前循环，继续下一次循环
3. 标号：可以在循环前面起名字，break时指定名字
4. 无限循环搭配break/无限循环的三种格式
	1. while（true）{}
	2. for(;;){}
	3. do{}while(true)
