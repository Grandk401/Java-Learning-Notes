# 概念
1. JRE：***Java Runtime Enviornment***==Java运行环境==，包含JVM和Java核心类库
2. 类：Java文件在代码中的一个集合体现
3. 类库：存放多个Java类的仓库
4. 开发过程：编写代码、编译代码、运行代码 
5. Java文件属于源文件，无法被JVM识别，需要使用==JDK==翻译为JVM可识别的再运行
6. JDK：***Java Development Kit***是Java语言的==软件开发工具包==，内部包含了JRE 和代码的编译工具和运行工具等。
